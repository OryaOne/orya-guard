# Code of Conduct

## Our Standard

This project is committed to a respectful, professional, and welcoming community.

Examples of positive behavior:

- being respectful and constructive
- assuming good intent
- giving actionable technical feedback
- keeping discussions focused on the work

Examples of unacceptable behavior:

- harassment, discrimination, or intimidation
- personal attacks or insulting language
- bad-faith disruption of discussions or reviews
- publishing private information without permission

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests, discussions, and review threads.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or contributions that do not align with this Code of Conduct.

## Reporting

To report unacceptable behavior, open a private channel with the maintainer through GitHub.
